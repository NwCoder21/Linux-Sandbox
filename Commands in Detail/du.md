# `du` - disk usage

This command is used to display the size of a filee or directory.

## Syntax:
```yaml
$ du <option> <file/directory> 
```
![image](https://user-images.githubusercontent.com/107522496/196989134-4e01ecf3-1247-4987-8e1a-64398bcb3ab6.png)
